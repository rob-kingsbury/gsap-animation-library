Scroll

Multi-layer depth

The parallax effect creates a sense of depth by moving background elements at different speeds. Each layer responds to scroll position, creating an immersive 3D-like experience.

Immersive experiences

Background images that move at different rates create a cinematic scroll effect.

Image parallax

Apply the parallax effect to background images for a dramatic, cinematic feel. The image shifts as you scroll, creating depth and visual interest.

Code Reference

// Multi-layer parallax with different speeds
gsap.utils.toArray('[data-speed]').forEach(layer => {
    const speed = layer.dataset.speed;

    gsap.to(layer, {
        y: () => window.innerHeight * speed * -1,
        ease: 'none',
        scrollTrigger: {
            trigger: '.parallax-hero',
            start: 'top top',
            end: 'bottom top',
            scrub: true
        }
    });
});
// Image parallax effect
gsap.to('.image-layer img', {
    y: '-20%',  // Move up as we scroll down
    ease: 'none',
    scrollTrigger: {
        trigger: '.image-parallax-hero',
        start: 'top top',
        end: 'bottom top',
        scrub: true
    }
});
// Hero content entrance animation
