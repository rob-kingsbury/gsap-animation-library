Hamburger Menu

Click the hamburger icon in the header to see the full-screen menu animation. Below are various hamburger transformation styles.

Hamburger Transformation Styles

Classic X

Lines rotate to form X

Arrow

Transforms into arrow

Squeeze

Middle line fades out

Spin

Rotates while morphing

Elastic

Bouncy transformation

Minus

Collapses to single line

Code Reference

// Classic X transformation
const tl = gsap.timeline({ paused: true });

tl.to('.line-1', {
    y: 12,           // Move to center
    rotation: 45,    // Rotate 45 degrees
    duration: 0.3
})
.to('.line-2', {
    opacity: 0,       // Fade out middle line
    duration: 0.1
}, 0)
.to('.line-3', {
    y: -12,          // Move to center
    rotation: -45,   // Rotate -45 degrees
    duration: 0.3
}, 0);

// Toggle on click
hamburger.addEventListener('click', () => {
    isOpen ? tl.reverse() : tl.play();
    isOpen = !isOpen;
});
// Full-screen menu reveal
const menuTl = gsap.timeline({ paused: true });

// Circle clip-path reveal
menuTl.to('.menu', {
    clipPath: 'circle(150% at calc(100% - 52px) 40px)',
    duration: 0.8,
    ease: 'power4.inOut'
})
// Stagger in menu items
.from('.menu-nav a', {
    opacity: 0,
    y: 50,
    stagger: 0.1,
    duration: 0.5,
    ease: 'power3.out'
}, '-=0.4');
// Alternative: Slide reveal from right
const slideTl = gsap.timeline({ paused: true });

slideTl.to('.menu', {
    x: 0,            // Slide from right (start at x: 100%)
    duration: 0.5,
    ease: 'power3.inOut'
})
.from('.menu-nav li', {
    x: 100,
    opacity: 0,
    stagger: 0.08,
    duration: 0.4
}, '-=0.2');
